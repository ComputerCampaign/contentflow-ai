# 数据目录

此目录用于存储爬虫抓取的数据。

## 目录结构

```
data/
  └── task_id/
      ├── images/
      │   ├── page_name_1.jpg
      │   ├── page_name_2.png
      │   └── ...
      ├── metadata/
      │   ├── images.csv
      │   └── page_info.json
      └── page.html
```

每个爬取任务都有一个唯一的任务ID，所有相关数据都存储在以任务ID命名的目录中。

## 注意事项

- 此目录中的数据通常不会被版本控制系统跟踪
- 请确保此目录有足够的磁盘空间用于存储图片
- 如果需要清理数据，可以删除此目录中的内容，但请保留目录结构
